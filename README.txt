Project Title:
Finance

Summary:
In this project, I had the task of creating a Java application that theoretically simulates a banking transaction, without using real accounts. All data is fictionally created in the application, and the transaction itself is nothing more than a simple addition or subtraction operation.


Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.


Prerequisites and Installing:
To run my sample you need to install the following ->
-> Java version 8 : https://java.com/en/download/help/download_options.xml
->Install what java ide you like
